Exercícios de fixação

Curso HTML - Aula 03

Exercício 1

  • Crie um arquivo chamado exercicio_01.html
  • Adicione um título e um subtítulo para a página
  • Adicione um link que direciona para o site da acadix.com.br
  • Faça com que esse link abra em uma nova guia

Exercício 2

  • Crie um arquivo chamado exercicio_02.html
  • Adicione um título e um subtítulo para a página
  • Adicione um link que direciona para o exercicio_01.html
  • Crie um link no exercicio_01.html para apontar para o exercicio_02.html
  • Coloque ambos os links em uma lista não ordenada

Exercício 3

  • Crie um arquivo chamado exercicio_03.html
  • Adicione um título e um subtítulo para a página
  • Adicione um link que direciona para o exercicio_01.html
  • Adicione um link que direciona para o exercicio_02.html
  • Adicione uma imagem no lugar do texto que direciona para o exercicio_01.html
  • Adicione um botão no lugar do texto que direciona para o exercicio_02.html
  • Repita o processo nos demais arquivos

Exercício 4

  • Adicione um link que direciona para o exercicio_01.html

Exercício 5
