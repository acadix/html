Exercícios de fixação

Exercício 1

  • Crie um arquivo chamado exercicio_01.html
  • Adicione um título e um subtítulo para a página
  • Crie um formulário que utilize o método GET
  • Adicione um campo do tipo "text" para o nome
  • Adicione um campo de tipo "text" para o sobrenome
  • Adicione um input do tipo "submit"
  • Utilize a propriedade "action" da tag "form" e a API PostBin para testar o envio do seu formulário

Exercício 4

  • Crie um arquivo chamado exercicio_04.html
  • Adicione um título e um subtítulo para a página
  • Crie um formulário que utilize o método POST
  • Utilizando o input do tipo "checkbox", crie um cardápio com 5 opções de refeições
  • Adicione um campo do tipo "radio" com as opções: "Entregar" e "Buscar no local"
  • Adicione campos de texto para Rua, bairro e cidade
  • Adicione um campo numérico para o número da casa
  • Adicione um campo do tipo "range" com um mínimo de 0 e máximo de 5 para selecionar a quantidade de estrelas que dará para este restaurante
  • Adicione um campo do tipo "color" para selecionar a cor da embalagem da entrega
  • Organize seu formulário utilizando a tag "fieldset"
  • Adicione um input do tipo "submit"
  • Utilize a propriedade "action" da tag "form" e a API PostBin para testar o envio do seu formulário

Exercício 5

  • Crie um arquivo chamado exercicio_05.html
  • Adicione um título e um subtítulo para a página
  • Crie um formulário que utilize o método POST
  • Crie um "fieldset"
  • Adicione uma tag "legend" com o texto: Criar um novo post no blog
  • Adicione um campo do tipo "file" para o usuário enviar uma imagem
  • Lembrese de utilizar o "enctype" na tag "form"
  • Adicione um campo "text" para o título
  • Adicione uma tag "textarea" para o conteúdo
  • Adicione um "select" com duas "options": (destaque), (normal)
  • Configure o "select" para que inicie com um valor vazio e seja obrigatório selecionar uma das duas opções
  • Adicione um input do tipo "submit"
  • Lembre-se de que a API do PostBin não aceita requisições com arquivos, então não precisa testar o envio do seu formulário
