Exercícios de fixação

Exercício 1

  • Crie um arquivo chamado exercicio_01.html
  • Adicione um título e um subtítulo para a página.
  • Crie uma meta tag que vai ajudar os dispositivos móveis a fazer com que a largura da página seja igual à largura do dispositivo.
  • Crie uma meta tag para definir o charset da sua página.
  • Crie uma meta tag para descrever o conteúdo da sua página.
  • Crie uma meta tag para definir as palavras-chave da sua página.
  • Crie uma meta tag para definir o autor da página.
  • Crie uma meta tag para informar se você quer que sua página seja indexada pelos mecanismos de busca.

Exercício 2

  • Crie um arquivo chamado exercicio_02.html
  • Adicione um título e um subtítulo para a página.
  • Siga os mesmos passos do exercício anterior.
  • Adicione uma tag OG para definir o título que sera exibido em uma prévia do seu link em redes sociais
  • Adicione uma tag OG para definir a descrição que será exibida em uma prévia do seu link em redes sociais
  • Adicione uma tag OG para definir a imagem que será exibida em uma prévia do seu link em redes sociais
  • Adicione uma tag OG para definir a URL que será exibida em uma prévia do seu link em redes sociais
  • Adicione uma tag OG para definir o tipo de conteúdo que será exibido em uma prévia do seu link em redes sociais

Exercício 3

  • Crie um arquivo chamado exercicio_03.html
  • Adicione um título e um subtítulo para a página.
  • Adicione um parágrafo descrevendo com suas palavras o que é o HTML. Inicie seu parágrafo com o seguinte texto: "HTML (HyperText Markup Language) é uma linguagem de marcação utilizada...
  • Envolva a palavra HTML no seu texto com a tag "acronym". Utilize a propriedade "title" para descrever o significado do acrônimo HTML.
  • Envolva o trecho "HyperText Markup Language" com a tag "span". E adicione a propriedade "lang" para definir o idioma do texto.
  • Adicione uma imagem com o logo da Acadix. Utilize a propriedade "alt" para descrever o conteúdo da imagem.
  • Adicione um link para o site da Acadix. utilize a propriedade "title" para descrever o destino do link.
  • Adicione uma tabela com 3 colunas e 3 linhas. Utilize a propriedade "scope" para definir os títulos de cada coluna.
  • Utilize tags semânticas para deixar trechos do seu texto em negrito e itálico.
  • Separe seu código em 3 tags semânticas: "header", "content" e "footer".
  • Adicione uma tag "nav" com links para navegar na sua página entre os elementos. Utilize a propriedade "role" para definir o papel da tag "nav" e a propriedade "aria-label" para descrever o conteúdo da tag "nav". Utilize nos links a propriedade "title" para descrever o destino.
