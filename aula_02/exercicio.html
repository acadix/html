Exercícios de fixação

Curso HTML - Aula 02

Exercício 1

  • Crie um arquivo chamado exercicio_01.html
  • Adicione um título e um subtítulo para a página
  • Crie uma lista ordenada com pelo menos 5 itens relacionados a temas de sua escolha.
  • Incorpore um link para uma página externa na sua lista não ordenada. Certifique-se de que o link esteja funcional.
  • Adicione uma lista dentro de um dos itens da sua lista ordenada. Essa lista interna deve conter pelo menos 3 itens.
  • Marque um dos itens de sua lista com um traço, para demonstrar que ele foi riscado da lista.

Exercício 2

  • Crie um arquivo chamado exercicio_02.html
  • Adicione um título e um subtítulo para a página
  • Adicione links para navegar entre os arquivos exercicio_01 e exercicio_02
  • Crie uma lista ordenada utilizando marcadores romanos
  • Adicione um parágrafo de texto antes de cada uma das suas listas, explicando brevemente o conteúdo delas.

Exercício 3

  • Crie um arquivo chamado exercicio_03.html
  • Adicione um título e um subtítulo para a página
  • Adicione links para navegar entre os outros arquivos de exercícios
  • Crie uma tabela com 3 linhas e 3 colunas
  • Adicione um cabeçalho a sua tabela
  • Adicione uma linha de rodapé a sua tabela, com uma célula que ocupe as 3 colunas

Exercício 4

  • Crie um arquivo chamado exercicio_04.html
  • Adicione um título e um subtítulo para a página
  • Adicione links para navegar entre os outros arquivos de exercícios
  • Crie uma tabela com 3 linhas e 3 colunas, com bordas e espaçamento entre as células
  • Adicione um cabeçalho a sua tabela e siga os seguintes requisitos:
    1. A primeira célula com o texto "Exercício"
    2. A segunda célula com o texto "Descrição"
    3. A terceira célula com o texto "Tempo gasto"
  • Preencha a tabela com os dados dos exercícios que você fez até agora
  • Adicione uma linha de rodapé a sua tabela, com uma célula que ocupe as 3 colunas
  • Adicione uma nova célula a sua tabela, com o texto "Repetir", ela deve ocupar a primeira coluna e todas as linhas da tabela. (lembre-se do atributo "rowspan")
